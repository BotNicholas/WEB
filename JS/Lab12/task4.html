Задание IV

    Торговый агент продает пять различных продуктов, цены которых следующие: 
    продукт1 - $2,98; 
    продукт2 - $4,50; 
    продукт3 - $9,98; 
    продукт4 - $4,49; 
    и продукт5 - $6,87.

    Создайте скрипт, который считывает пару чисел : 
    a) Номер продукта (от 1 до 5) 
    b) Проданное количество

    Программа должна использовать оператор switch, который будет определять и выводить на 
    HTML-страницу цену каждого продукта, а также вычислять общую стоимость проданных продуктов. 
    Используйте функцию prompt для ввода значений. Ввод значений должен завершаться вводом 
    значения 0 для номера продукта.
